# 💻 Instalar Visual Studio Code

1. [Acceder a la página oficial de Visual Studio Code: https://code.visualstudio.com/](https://code.visualstudio.com/)
2. Hacer clic en el botón de descarga para tu sistema operativo (Windows, macOS o Linux).
3. Seguir las instrucciones de instalación proporcionadas por el instalador.
4. Una vez instalado, abrir Visual Studio Code.
5. Para instalar la extensión de Python, hacer clic en el ícono de extensiones en la barra lateral izquierda (o presionar `Ctrl+Shift+X`).
6. Buscar "Python" en la barra de búsqueda de extensiones.
7. Hacer clic en "Instalar" en la extensión de Python desarrollada por Microsoft.
8. Después de la instalación, reiniciar las extensiones de Visual Studio Code para que los cambios surtan efecto.
